Stores items, such as request handlers, using an HTTP method and a path as keys.

Paths can be static ("path/to/item") or variable ("users/:group/:user") and contain wildcards ("path/*/item/*").

Variables (starting with :) will match whatever word the request path contains at that point and it will be sent as a value to the item.

Wildcards (a single *) will consume the segments until the rest of the path gives a match.

pattern = "a/*/b"
"a/c/b" -> match
"a/c/d/b" -> match
"a/b" -> no match
"a/c/b/d" -> no match

pattern = "a/b/*"
"a/b/c" -> match
"a/b/c/d" -> match
"a/b" -> no match
